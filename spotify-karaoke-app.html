Anleitung für Entwickler

Da diese App direkt in deinem Browser läuft, musst du sie einmalig bei Spotify registrieren.

  1. 1
    Öffne das Spotify Dashboard und logge dich ein.
  2. 2
    Klicke auf "Create App". Name: "My Karaoke".
    Redirect URI: Kopiere die Adresse aus dem Login-Tab (blauer Kasten).
  3. 3
    Klicke in deiner neuen App auf Settings. Kopiere die Client ID und füge sie im Login-Tab ein.

warning Wichtige Hinweise

  • check Du benötigst Spotify Premium.
  • check Wenn du die App lokal als Datei öffnest (file://), funktionieren Redirects oft nicht. Nutze besser einen lokalen Server (z.B. VS Code Live Server).
  • check Starte Spotify auf deinem Gerät, bevor du dich hier verbindest.
Dies ist eine Client-Side Only App. Keine Daten werden an Server gesendet außer an Spotify direkt.
